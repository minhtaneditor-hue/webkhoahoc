# 🚀 MINH TAN ACADEMY // LMS ELITE PLATFORM

Welcome to the command center of **Minh Tan Academy**. This is a high-authority, premium LMS ecosystem designed for high-ticket video coaching and digital asset distribution.

## ✨ Premium Features Included

### 🤖 AI Agentic Chat Assistant
- Integrated `AIChatAssistant` for real-time student support and automated sales funnels.
- Minimalist, premium UI with glassmorphism and motion-blur interactions.

### 🛡️ SaaS DRM Video Integration
- High-impact, full-screen video hero section integrated into the landing page footer.
- Showcases the "Datacore v3.2" security system with professional-grade cinematography.
- Built-in brand trust components using curated typography (Instrument Serif, Manrope).

### ⚡ Automated CRM Operations
- **God Mode Admin**: Secure, dynamic access control for authorized command centers (`minhtaneditor@gmail.com`).
- **Content Master**: Full control over courses, pricing, and video curriculum deployment.
- **Retention CRM**: Integrated payment tracking (VNPay) and user activity monitoring.

### 🔄 One-Touch Deployment
- Custom automated workflow for zero-friction updates.
- Use `sh deploy.sh "commit message"` to sync everything to GitHub and Vercel in seconds.

## 🛠️ Stack & Technologies
- **Framework**: Next.js 15 (App Router)
- **Database/Auth**: Supabase (PostgreSQL + JWT)
- **Styling**: Tailwind CSS 4 + Framer Motion (Rich Interactions)
- **Typography**: Google Fonts (Instrument Serif, Manrope, Cabin, Inter)

## 🏗️ Getting Started

### 1. Configure Environment Variables
You MUST set these variables in your `.env.local` (and Vercel Dashboard):
```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
SUPABASE_SERVICE_ROLE_KEY=your_key
NEXT_PUBLIC_ADMIN_EMAIL=minhtaneditor@gmail.com
```

### 2. Launch Local Lab
```bash
npm run dev
```

### 3. Deploy to Mission Control
```bash
npm run deploy "Brief description of changes"
```
